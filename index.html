4 · Faithfulness diagnostics

An attribution is only useful if it tracks the model. These tests perturb inputs toward the background baseline and check whether φ predicted the resulting change in f̂.

Deletion & insertion curves (Petsiuk et al., 2018)

Deletion: features are removed (set to baseline) in decreasing |φ| order — a faithful explainer produces a steep early drop, i.e. low AUC. Insertion is the reverse: high AUC is better. Random-order curves are shown as a null reference.

Scorecard

Faithfulness correlation (Bhatt et al., 2020): Pearson r between Σi∈S φi and f̂(x) − f̂(xS→baseline) over random subsets S. IG completeness gap: |ΣφIG − (f̂(x) − f̂(baseline))|, which should be ≈ 0 by construction. Rank agreement: mean Spearman ρ between per-instance SHAP and IG orderings — a consistency (sanity) check between two independent estimators, not proof of correctness.

Methodology notes

Kernel SHAP

For a coalition z ∈ {0,1}M, the value function is v(z) = Eb~B[f̂(xz ⊕ b1−z)], estimated by averaging over the background set B. Shapley values are the solution of a weighted least-squares problem with kernel weight π(z) = (M−1)/(C(M,|z|)·|z|·(M−|z|)), subject to the efficiency constraint Σφ = f̂(x) − E[f̂]. This app enumerates all 2M−2 proper coalitions when M ≤ 11 (exact under the interventional value function) and otherwise uses paired coalition sampling under the kernel distribution.

Integrated Gradients

IGi(x) = (xi − ẋi) ∫01 ∂f̂(ẋ + α(x − ẋ))/∂xi dα, approximated with a 64-step Riemann sum via tf.js autodiff, baseline ẋ = training mean. The completeness axiom gives an internal consistency check reported on the Diagnose tab.

Reading the diagnostics

Deletion/insertion AUCs are comparative, not absolute: judge them against the random-order null shown in grey. Faithfulness correlation is sensitive to the same off-manifold evaluation problem as all baseline-replacement tests; the benchmark generator exists precisely so the pipeline can be validated where ground truth is known before it is trusted on real data.
